# Keyboard-Controlled System Shutdown (Educational Project):

# Overview:

This project is a **Python-based system utility** that listens for a keyboard shortcut and performs a system shutdown.

 This is built strictly for **educational purposes** to understand:

* Keyboard event handling in Python
* System command execution
* Startup automation in operating systems



# Features:

*  Global hotkey detection (`keyboard` module)
*  Instant system shutdown on key press
*  Runs on system startup (optional)
*  Lightweight and easy to modify



# Tech Stack:

* Python 3
* `keyboard` library
* OS system commands






# Usage:

Run the script:


keyshotd.py


Then press:


S key → Shutdown system



# Important Notes:

* Run as **Administrator** (required for keyboard detection)
* Script is **not hidden** and runs only when started
* Designed for **learning and personal use only**
* Do not use on systems without user knowledge or consent



# Learning Outcomes:

* Handling keyboard events in Python
* Interacting with operating system commands
* Basic automation techniques
